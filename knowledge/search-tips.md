# 🔍 搜索技巧汇总

> 整理者：✨ 小琳 | 更新于 2026-02-05

## 通用搜索技巧

### 关键词优化
- 加引号精确匹配：`"完整短语"`
- 排除词：`关键词 -排除词`
- 限定网站：`site:github.com 关键词`
- 限定文件类型：`filetype:pdf 关键词`

## 特殊场景

### 搜索微信公众号内容
普通搜索引擎搜不到微信公众号文章，用**搜狗搜索**：

- 网址：https://weixin.sogou.com/
- 直接搜索公众号名称或文章关键词

### 搜索影视资源
技巧：加上「更新至 XX 集」能找到最新资源

```
夜色正浓 更新至12集 夸克
年少有为 全集 网盘
```
