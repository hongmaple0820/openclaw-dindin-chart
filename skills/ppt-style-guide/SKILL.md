# PPT 风格指南

用审美偏好而非模板来定义 PPT 风格。

## 核心理念

> 模板是死的，审美是活的。
> 
> 与其收集 100 个模板，不如建立清晰的审美标准。

## 激活场景

- 用户说"做 PPT"、"演示文稿"、"slides"
- 需要将内容转换为演示格式
- 用户想定义自己的 PPT 风格

## 工作流程

### 1. 内容准备 → NotebookLM

将原始内容（文档、笔记、想法）输入 NotebookLM，让它：
- 提取核心观点
- 生成结构大纲
- 建议每页要点

### 2. 风格定义 → 审美偏好

不是选模板，而是定义偏好：

```yaml
# 我的 PPT 审美偏好

## 整体风格
氛围: 简约专业 / 活泼创意 / 科技感 / 学术严谨
色调: 冷色系 / 暖色系 / 单色 / 高对比

## 排版
每页文字量: 极少（3-5 行）/ 适中（5-8 行）/ 较多
字体风格: 无衬线现代 / 衬线经典 / 手写风
对齐方式: 左对齐 / 居中 / 非对称

## 视觉元素
图片风格: 实拍 / 插画 / 图标 / 抽象
图表风格: 极简 / 数据密集 / 信息图
留白程度: 大量留白 / 适中 / 紧凑

## 动画
过渡效果: 无 / 简单淡入 / 动态
元素动画: 无 / 逐条出现 / 复杂动画
```

### 3. 生成与迭代

将审美偏好转换为具体指令：

```markdown
请按以下风格生成 PPT：

**整体**：简约专业，冷色调（主色 #2563EB）
**排版**：每页不超过 5 行文字，无衬线字体，左对齐
**视觉**：使用线条图标，大量留白
**图表**：极简风格，去除网格线和多余装饰
```

## 审美偏好库

### 风格 A：极简主义

```yaml
名称: 极简主义
适用: 高端品牌、设计公司、创意提案

特点:
  - 大量留白（页面 60%+）
  - 每页只讲一件事
  - 单色或双色
  - 无装饰元素
  - 大字号标题
  
配色示例:
  - 黑白灰
  - 白 + 一个强调色
  
字体: Helvetica, SF Pro, 思源黑体
```

### 风格 B：数据驱动

```yaml
名称: 数据驱动
适用: 商业报告、财务汇报、运营分析

特点:
  - 图表为主角
  - 数字突出显示
  - 结论在最上方
  - 专业配色
  
配色示例:
  - 深蓝 + 浅蓝 + 灰
  - 深绿 + 浅绿 + 白
  
图表原则:
  - 去除图例，直接标注
  - 突出关键数据
  - 添加趋势线和注释
```

### 风格 C：故事叙事

```yaml
名称: 故事叙事
适用: 产品发布、演讲、TED 风格

特点:
  - 全屏图片背景
  - 大标题覆盖
  - 每页一个情绪
  - 强调故事弧
  
配色示例:
  - 随图片而变
  - 文字用白色或黑色确保可读
  
结构:
  - 开场：提出问题/矛盾
  - 发展：展示旅程
  - 高潮：揭示解决方案
  - 结尾：号召行动
```

### 风格 D：学术报告

```yaml
名称: 学术报告
适用: 论文答辩、研究报告、技术分享

特点:
  - 结构清晰
  - 引用规范
  - 图表严谨
  - 适量文字说明
  
配色示例:
  - 白底 + 深蓝/深灰
  - 低饱和度配色
  
必备元素:
  - 页码
  - 章节标题
  - 参考文献
```

## 偏好积累方法

### 1. 记录喜欢的

看到好的 PPT 时，记录：
- 为什么觉得好？
- 哪个元素打动我？
- 适合什么场景？

```markdown
## 2024-02-06 发现

来源: [某产品发布会]
喜欢的点:
- 每页只有一句话 + 一张大图
- 数字用 200pt 超大字号
- 过渡用简单的淡入淡出

适用场景: 产品发布、愿景演讲
```

### 2. 记录不喜欢的

同样重要：
- 这个设计哪里让我不舒服？
- 怎么改会更好？

```markdown
## 避免

❌ 每页塞满文字
❌ 使用剪贴画
❌ 花哨的过渡动画
❌ 多于 3 种颜色
❌ 复杂的图表装饰
```

### 3. 定期提炼

每月回顾，更新审美标准：
- 哪些偏好是稳定的？
- 有没有新发现？
- 针对不同场景的区分

## 最终输出

一份真正属于你的 PPT 风格指南：

```markdown
# [你的名字] 的 PPT 审美标准

## 核心原则
1. 少即是多
2. 每页一个核心观点
3. 视觉服务于内容

## 配色偏好
- 主色：#2563EB（科技蓝）
- 辅色：#64748B（中性灰）
- 强调：#F59E0B（温暖橙）

## 字体偏好
- 标题：思源黑体 Bold
- 正文：思源黑体 Regular
- 数据：DIN / Roboto Mono

## 布局偏好
- 左对齐
- 标题在左上
- 大量右侧留白

## 禁区
- 不用渐变背景
- 不用阴影效果
- 不用超过 3 种颜色
- 不用动画（除非必要）
```
