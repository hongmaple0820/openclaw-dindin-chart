# PPT 审美偏好指南

制作 PPT 时的设计规范和审美偏好。

---

## 🎨 配色方案

### 主题色

| 颜色 | 色值 | 用途 |
|:-----|:-----|:-----|
| 主色 | `#2563EB` | 标题、重点 |
| 成功 | `#10B981` | 正向信息 |
| 警告 | `#F59E0B` | 注意事项 |
| 错误 | `#EF4444` | 问题、风险 |
| 粉色 | `#F472B6` | 小猪相关 🐷 |

### 背景色

| 场景 | 颜色 |
|:-----|:-----|
| 浅色背景 | `#FFFFFF` / `#F8FAFC` |
| 深色背景 | `#1E293B` / `#0F172A` |
| 代码块 | `#F1F5F9` |

---

## 📝 字体

### 推荐字体

| 用途 | 中文 | 英文 |
|:-----|:-----|:-----|
| 标题 | 思源黑体 | Inter / Helvetica |
| 正文 | 微软雅黑 | -apple-system |
| 代码 | - | Fira Code / JetBrains Mono |

### 字号

| 元素 | 字号 |
|:-----|:-----|
| 主标题 | 48px |
| 副标题 | 32px |
| 正文 | 24px |
| 注释 | 18px |

---

## 📐 布局原则

### 留白

- 边距至少 **10%** 页面宽度
- 元素间距保持一致
- 不要填满整个页面

### 对齐

- 统一使用左对齐或居中对齐
- 避免混合对齐方式
- 标题居中，正文左对齐

### 层次

1. 标题最突出
2. 重点内容次之
3. 辅助信息最小

---

## 📊 图表风格

### 颜色搭配

```
主要: #2563EB (蓝)
次要: #10B981 (绿)
第三: #F59E0B (黄)
第四: #F472B6 (粉)
```

### 图表规范

- 使用圆角矩形
- 线条粗细 2px
- 避免 3D 效果
- 图例放右侧或下方

## ❌ 避免的做法

- 🚫 过多颜色（超过 5 种）
- 🚫 密密麻麻的文字
- 🚫 花哨的动画
- 🚫 不一致的字体
- 🚫 低分辨率图片
- 🚫 无意义的装饰

## 🛠️ 工具

### Marp CLI

```bash
# 安装
npm install -g @marp-team/marp-cli

# 生成 PPTX
marp --no-stdin slides.md --pptx -o slides.pptx

# 生成 PDF
marp --no-stdin slides.md --pdf -o slides.pdf
```

### Marp 模板

```markdown
---
marp: true
theme: default
paginate: true
---

# 标题页

副标题

---

# 第二页

- 要点 1
- 要点 2

---
```
